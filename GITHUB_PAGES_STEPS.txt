GitHub Pages deployment steps

1. Create a new public GitHub repository.
2. Upload these files to the repository root:
   - index.html
   - MyPage.html
   - brick.html
   - spaceShoot.html
   - maze.html
   - PVZ2Floors.html
   - backgroundMusic.mp3
   - PVZBackgroundMusic.mp3
3. In GitHub, open Settings -> Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save.
6. Wait about 1-2 minutes.
7. Your playable link will be:
   https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/

Direct game links will be:
- https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/brick.html
- https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/spaceShoot.html
- https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/maze.html
- https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/PVZ2Floors.html
